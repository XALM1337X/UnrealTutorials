CoopGame is the final result of Unreal Engine 4 Mastery: Create Multiplayer Games with C++ course created by Tom Looman of Epic Games.

This game isnt graphically detailed, but does show comprehensive backend C++ knowledge along with blueprint C++ coupling, and displays fully functioning client/server RPC systems for proper multiplayer replication. Also shows AI/Advanced AI tree comprehension along with proper GameMode/GameState replication to server/clients.
